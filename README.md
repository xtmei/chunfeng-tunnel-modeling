# 春风隧道结构科普示意动画

一个可直接本地运行的 Three.js 科普网页，用于展示深圳春风隧道“单洞双层”结构逻辑。

## 运行方式

1. 直接双击 `index.html` 用浏览器打开（推荐 Chrome / Edge 最新版）。
2. 或使用本地静态服务器（避免部分浏览器模块策略限制）：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000`。

## 交互说明

- **播放 / 暂停**：控制自动分镜播放。
- **上一幕 / 下一幕**：手动切换五个场景。
- **进度条**：拖动到任意时间点。
- **显示标签**：开关结构文字标注。

## 主要可修改项

在 `main.js` 顶部集中调整：

- `DIMENSIONS`：隧道外径、内径、开挖直径、示意长度。
- `COLORS`：主色、强调色、结构材质色。
- `scenes`：每一幕文案与时长。

结构组件由程序化几何生成（无外部模型），`buildCrossSection()` 中可继续细化构件尺寸与位置。
